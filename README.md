# uuid01's Personal Site

个人 / Personal portfolio website

技术主页## 简介

这是一个展示个人简介、项目和生活点滴的个人网站。

## 技术栈

- HTML5
- CSS3 (纯手写动画效果)
- JavaScript (原生)
- Google Fonts (JetBrains Mono, Noto Sans SC, Orbitron)

## 页面结构

- `index.html` - 首页，展示个人简介和技能标签
- `projects.html` - 项目页面
- `life.html` - 生活点滴
- `life-detail.html` - 生活详情页
- `css/style.css` - 样式文件
- `js/main.js` - 交互脚本
- `favicon.svg` - 网站图标

## 本地运行

直接在浏览器中打开 `index.html` 即可预览。
