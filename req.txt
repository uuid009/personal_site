### 四、技术要求（非常重要）

网站将部署到 **GitHub Pages**，因此需要注意：

1. **不要依赖复杂构建工具**

   * 不使用 Node build 工具（如 webpack / vite）
   * 优先使用 **纯 HTML + CSS + JavaScript**

2. **CSS 与 JS 必须使用相对路径**

例如：

```
./css/style.css
./js/main.js
```

避免 GitHub Pages 出现加载失败问题。

3. 项目目录结构建议如下：

```
/project-root
│
├── index.html
├── projects.html
├── life.html
│
├── css/
│   └── style.css
│
├── js/
│   └── main.js
│
└── assets/
    └── images/
```

4. 所有页面都能直接在 GitHub Pages 正常访问。
